CodeIgniter框架

Explain project files:
  這個專案我們使用CI(CodeIgniter)框架。登入的部分，我們導入Google登入，而首頁的地圖是使用google map。
當使用者進入首頁後，就會看到其他人在任以點上發的文章，使用者可以在其他人或自己發的文章下做評論。當使用者想
發文時，只要在任意點上點擊，便會跳出輸入視窗，此視窗是用Jquery AIP，使用者必須上傳圖片才可以發文，上船的部分是用了
Jquery Upload API，當使用者輸入完並按下確定，該發文便會出現在剛剛點擊的點上。
